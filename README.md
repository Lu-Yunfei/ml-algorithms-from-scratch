# Machine Learning Algorithms Implementation

Course assignments for DDA3020 Machine Learning at CUHKSZ (Spring 2026).
Completed key algorithm implementations based on course framework, including core math derivations and programming tasks.

## Projects

### HW1: Gradient Descent & Logistic Regression
- Implemented gradient descent to minimize a 2D function, visualized optimization trajectories with contour plots
- Experimented with different learning rates (0.01, 0.1, 1.1) and analyzed convergence behavior
- Built multi-class logistic regression with L1/L2 regularization on Iris dataset using gradient descent
- **Tech**: Python, NumPy, Matplotlib

### HW2: Decision Tree & Neural Network
- Compared Decision Tree vs Random Forest (n_estimators ∈ {10, 50, 100}) on Breast Cancer dataset
- Built a Multi-Layer Perceptron with PyTorch featuring a 2D bottleneck layer to visualize learned feature space
- Achieved >95% test accuracy on binary classification task
- **Tech**: Python, PyTorch, scikit-learn, Matplotlib

### HW3: K-Means & PCA (In Progress)
- Implementing K-Means clustering from scratch with elbow method and silhouette analysis
- PCA for dimensionality reduction and noise filtering (Eigenfaces)
- **Tech**: Python, NumPy, scikit-learn
